# homebrew-notchout

Homebrew tap for [NotchOut](https://github.com/hoorangyee/notchout) — a macOS utility that surfaces menu bar icons hidden behind the MacBook notch.

## Install

```bash
brew install --cask hoorangyee/notchout/notchout
```

## Upgrade

```bash
brew upgrade --cask notchout
```

## Uninstall

```bash
brew uninstall --cask notchout
```
